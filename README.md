# Boa Compra - Shipping Fees Interface

A simple React web interface that allows a user to acess the best shipping budget for a given product.

### Requirements
- npm
- Nodejs 13.x
- Visual Studio Code (Optional)

### Technologies
- React
- Typescript
- MaterialUI

### Running the project
- npm install
- npm run start
